# Banking Program

A simple Python banking application that supports showing account balance, depositing money, and withdrawing money with validation checks.

## Features

- Display current account balance
- Deposit funds with negative amount validation
- Withdraw funds with overdraft and negative amount protection
- Menu-driven command line interface

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Abhishek10roy/Banking-Program.git
   cd Banking-Program
   ```
2. Run the application:
   ```bash
   python Banking.py
   ```
3. Follow the on-screen menu to view balance, deposit, withdraw, or exit.

## Requirements

- Python 3.x

## Notes

- The application maintains a simple in-memory balance during runtime.
- The balance resets each time the program is restarted.
